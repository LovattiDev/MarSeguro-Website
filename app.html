Informações Técnicas do App

Tecnologias Utilizadas e Documentação do Código

O aplicativo foi desenvolvido utilizando tecnologias modernas da plataforma Android, com foco em desempenho, compatibilidade e estabilidade. Foram utilizadas bibliotecas amplamente adotadas no ecossistema Android, além de ferramentas que auxiliam na construção da interface, no gerenciamento das atividades do sistema e na realização de testes. Essas tecnologias formam a base que possibilita o funcionamento adequado do aplicativo em diferentes dispositivos e versões do sistema operacional.

Tecnologia Versão Detalhe
Android Gradle Plugin (AGP) 8.12.1 Responsável por compilar e buildar o app Android.
Java 11 Linguagem usada para desenvolver a lógica do app.
Android SDK (compileSdk) 36 Define a versão do Android usada para compilar o app.
Android SDK (minSdk) 24 Define a versão mínima do Android suportada.
AndroidX AppCompat 1.7.1 Garante compatibilidade com versões antigas do Android.
Material Design Components 1.12.0 Fornece componentes visuais seguindo o Material Design.
AndroidX Activity 1.10.1 Gerencia o ciclo de vida das telas (Activities).
ConstraintLayout 2.2.1 Permite criar layouts flexíveis e responsivos.
JUnit 4.13.2 Framework para testes unitários em Java.
AndroidX Test JUnit 1.3.0 Integra testes JUnit com o ambiente Android.
Espresso Core 3.7.0 Framework para testes automatizados de interface (UI).
